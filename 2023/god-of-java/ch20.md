- 질문 답안
    - `System.out.println` **메소드는 현업에서 절대 쓰지 말라고하는 메소드인데요. 그 이유가 무엇일까요?**

        ```java
        public void println {
        	newLine();  //critical section(임계영역)
        }
        
        protected void newLine {
        	synchronized {
        	
        	}
        }
        ```

        - 임계영역: 공유자원의 접근 순서에 따라 실행 결과가 달라지는 프로그램의 코드 영역이다.
        - synchronized 키워드가 있으면 멀티쓰레드 환경에서 A쓰레드가 newLine()을 실행하면 메소드는 잠기게된다. A쓰레드가 모두 사용하고 잠금을 풀어줘야 다른 쓰레드가 newLine() 메소드를 실행할 수 있다.
            - **오버헤드 발생**
            - 스프링을 실행하는 톰캣은 멀티쓰레드로 동작한다. 요청이 오면 스레드풀에서 쓰레드를 하나 가져와 요청을 처리한다. 그런데 `System.out.println()` 메서드를 여러 쓰레드에서 사용하면 그만큼 오버헤드가 발생하고 처리가 느려질 것이다.
